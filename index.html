代码编辑器用到Atom。

cmder命令行软件,cd /d E:\web，需要加上/d才能转到相应的目录。

创建一个本地服务器，让程序监听文件变化，atom保存时，自动刷新。

回到命令行，创建一个package.json文件。在命令行输入yarn init -y，回车；再去安装一个browser-sync；在命令行输入yarn add browser-sync，回车；

回到atom编辑器，打开package.json文件，在 "license": "MIT",后面加上以下内容： "scripts": { "start": "./node_modules/.bin/browser-sync start --server --no-notify --files='index.html, *.css, *.js'" },[然后再回到命令行，输入yarn start，回车。]
